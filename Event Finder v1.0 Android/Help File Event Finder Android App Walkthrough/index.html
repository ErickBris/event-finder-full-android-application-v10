Change the approriate Facebook ID upon facebook provided when you create apps via Facebook Developer Portal



H) AdMob Show/Hide - top



In Project Navigator Pane, Locate Config.java under com.config package.



Change either one of them true or false. By setting them all to true, all Ads will show on its corresponding views. By Default we set only the main view to show. :)
